2. Welche Daten gespeichert werden

  • Einstellungen & Inhalte (Übungen, Sets, Timer-Werte, Coach-/Stimm-Einstellungen, Radiosender, laufendes Workout) werden ausschließlich lokal auf deinem Gerät gespeichert (Browser-Speicher / localStorage). Sie verlassen dein Gerät nicht und werden nicht an uns übertragen.
  • Es findet keine Analyse, kein Tracking und keine Werbung statt.

3. Berechtigungen & Gerätefunktionen

  • Display anlassen (Wake Lock): Während eines laufenden Workouts kann der Bildschirm aktiv gehalten werden.
  • Sprachausgabe: Coach-Ansagen nutzen die Text-in-Sprache-Funktion deines Geräts (lokal). Es werden keine Sprachdaten an uns gesendet.
  • Audio: Signaltöne werden lokal erzeugt.

4. Drittanbieter-Dienste (optional)

Diese Dienste werden nur aktiv, wenn du sie selbst nutzt:

  • Spotify: Meldest du dich mit Spotify an, erfolgt die Anmeldung direkt bei Spotify (OAuth). Wiedergabe/Steuerung laufen über Spotify; es gelten die Datenschutzbestimmungen von Spotify. Ein Zugangstoken wird ggf. lokal auf deinem Gerät gespeichert, um angemeldet zu bleiben.
  • Radio-Streams: Wählst du einen Internet-Radiosender, baust du eine direkte Verbindung zum jeweiligen Stream-Anbieter auf (dessen Server erhält dabei technisch notwendige Verbindungsdaten wie deine IP-Adresse). Es gelten die Bestimmungen des jeweiligen Anbieters.

5. Hosting

Die Web-Version wird über GitHub Pages ausgeliefert. Beim Abruf der Seite verarbeitet GitHub technisch notwendige Server-Logdaten (z. B. IP-Adresse) gemäß GitHubs Datenschutzbestimmungen.

Da pixletics selbst keine personenbezogenen Daten auf einem eigenen Server speichert, findet darüber hinaus keine Verarbeitung durch uns statt.
